# PromptBin

PromptBin helps you save prompts you use often and paste them into your favorite AI chats with short, memorable shortcuts. Instead of repeatedly finding and copying a long prompt, type a shortcut such as `//email` and PromptBin inserts the saved text for you.

## Features

- Save and organize reusable prompts in one personal library.
- Create simple shortcuts for long or frequently used instructions.
- Use shortcuts directly in supported AI chat message boxes.
- Keep a shortcut visible while PromptBin sends the full prompt, or expand it in the message box before sending.
- Type `///shortcut` when you want to insert the full prompt immediately.
- Press Tab to expand shortcuts when that option is enabled.
- Quickly access recently used prompts from the extension popup.
- Customize the appearance of recognized shortcuts.
- Export a backup of your prompt library and import it again later.

## Supported AI chats

PromptBin works with:

- ChatGPT
- Claude
- Perplexity
- DeepSeek
- Google Gemini
- Grok
- Microsoft Copilot

## Privacy

Your prompts and preferences are stored locally in your browser. PromptBin only accesses the supported AI chat websites where its shortcut features need to work.

Removing the extension also removes its browser-stored data. Export a local backup first if you want to restore your prompt library after reinstalling PromptBin.

## Install from this folder

1. Open Chrome or Edge and go to the browser's Extensions page.
2. Turn on **Developer mode**.
3. Select **Load unpacked**.
4. Choose the PromptBin folder.

PromptBin opens your prompt library after installation so you can create your first shortcut.
